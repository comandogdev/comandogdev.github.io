3 minijuegos que puedes jugar inclinando tu móvil usando el acelerómetro.

Características
  • 1 jugador
  • 3 modos de juego
Tecnologías
  • Unity
  • Firebase
Plataformas
  • Windows
Retos encontrados
  • Monitorizar el acelerómetro del móvil con una interfaz web usando javascript
  • Guardar los datos del acelerómetro en Firebase
  • Acceder desde Unity a los datos del usuario y reflejarlo en en el gameplay
  • Emparejando el juego con la web que se usa como mando con un código QR
