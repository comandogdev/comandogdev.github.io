Juego en el que tienes que sortear obstáculos para poder escapar del IKEA.

Características
  • 1 jugador
Tecnologías
  • Unity
  • WebGL
Plataformas
  • Windows
  • Navegador
Retos encontrados
  • Obstáculos aleatorios
  • Cinemática final usando el motor
  • Si juegas desde el móvil con el navegador si giras la pantalla el juego se adapta.
  • Efecto de rebobinado usando efectos de post procesado
